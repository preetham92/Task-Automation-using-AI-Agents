# Task Automation using AI Agents

A full-stack platform for AI-powered travel expense management and claim automation.

## Features

- **Employee Portal:** AI-powered document upload and extraction for receipts/tickets.
- **Financier Dashboard:** Real-time analytics, processing metrics, and agent behavior logs.
- **Audit Trails:** Comprehensive system logs for every AI execution.

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS, Shadcn/UI, Lucide React.
- **Backend:** FastAPI (Python), AI Agents (Extraction & Validation).

## Getting Started

### Frontend Setup

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   Run the development server:
   ```

Bash

npm run dev
Backend Setup
Navigate to the backend directory.

Install requirements:

Bash

pip install -r requirements.txt
Run the FastAPI server:

Bash

uvicorn main:app --reload

---

### Final Push to Git

Now that you've added the README and checked the backend, run:

```bash
git add .
git commit -m "docs: add readme and final code cleanup"
git push origin main
```
